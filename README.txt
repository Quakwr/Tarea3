Instrucciones: Abrir una terminal dedicada para la carpeta en donde se contiene los archivos, en la terminal realizar:
1)    make
2)    make run

para hacerlo funcionar

Se utilizo java y javac version 11.

El juego es sencillo, consiste en recuperar todas las piezas de la nave antes de quedarse sin turnos, uno se puede encontrar con piezas, enemigos, pildoras y murallas. uno gana al recoger todas y pierde si se queda sin turnos.
